# Shino Tools

Tools that i made and reduce my usual workflow from over 50 clicks per outfit set down to fewer than 10

## Installation

### VRChat Creator Companion (VCC)
1. Add the package repository URL to your VCC list.
2. Add the **Shino VRC Tools** package to your target project.

## Tools:

### MassPackageImporter
- Reads a selected folder and imports every UnityPackage inside without PopUp
### QuickImport
- Easier Package import for Linux Systems
### VRCFuryToggleGenerator
- Automatically creates toggles for gameobjects
- Allows Possibility to assign global parameter
### VRCFuryGlobalParamCopier
- Meant to be used together with the Toggle Generator, copies the global parameters into an animation controller, and allows preset creation for VRChat's avatar parameter driver.


all tools are found in:

Tools -> Shino -> tool name
